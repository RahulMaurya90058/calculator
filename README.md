# 🧮 Calculator App

A simple and responsive calculator application built using HTML, CSS, and JavaScript. It performs basic arithmetic operations with a clean and user-friendly interface.

## 🚀 Live Demo

🔗 https://calculator-ga1x.onrender.com

## ✨ Features

- Addition, Subtraction, Multiplication, and Division
- Responsive and modern UI
- Real-time calculations
- Clear and Delete functionality
- Keyboard-friendly interface
- Lightweight and fast

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
├── index.html
├── style.css
├── script.js
└── README.md
```

## ▶️ How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/calculator-app.git
```

2. Navigate to the project directory:

```bash
cd calculator-app
```

3. Open `index.html` in your browser.
